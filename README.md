
# Gestion de Stock PHP

Application web de gestion de stock développée en PHP.

Fonctionnalités :
- gestion des produits
- gestion des catégories
- gestion des fournisseurs
- gestion des ventes
- authentification admin

Technologies :
- PHP
- MySQL
- HTML / CSS
- AdminLTE

Installation:

1. Cloner le projet
2. Copier le dossier dans htdocs
3. Importer la base de données dans phpMyAdmin
4. Lancer le projet avec XAMPP
